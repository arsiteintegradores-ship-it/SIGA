# SIGA

## Requisitos

- Python 3.11
- Dependencias en `requirements.txt`

## Instalación rápida (entorno local)

```bash
python -m pip install -r requirements.txt
python manage.py check
```

> Nota: si tu entorno no tiene acceso a PyPI (proxy o red restringida),
> instala las dependencias desde un mirror interno o usando un wheelhouse local.
